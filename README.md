# Garden Plans - Projets 3D pour le jardin

Modeles 3D parametriques (STL + plans techniques PDF) pour des meubles
de jardin a fabriquer soi-meme a partir de materiaux recycles.

## Projets

### Table basse en palettes recyclees

Repertoire : [`table_basse_palette/`](table_basse_palette/)

Table basse 100% bois de palette (1200 x 600 x 450 mm), inspiree du
projet [DIY Pallet Table](https://www.instructables.com/DIY-PALLET-TABLE-100-PALLET-WOOD/)
sur Instructables.

- Modele 3D (STL) + plan technique (PDF)
- Nomenclature detaillee avec guide de debit depuis 2 euro-palettes
- Guide de construction illustre pas a pas (12 pages)

## Pre-requis

```bash
pip install numpy numpy-stl matplotlib
```
